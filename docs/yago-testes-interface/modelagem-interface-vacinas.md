# Modelagem de Teste de Interface — Vacinas

Técnica utilizada

Particionamento de Equivalência.

Interface testada

Tela de Registro de Vacinas.

URL local:

http://localhost:3000/vacinas.html

Objetivo do teste

Verificar se a tela de vacinas valida corretamente os campos obrigatórios e o registro com dados válidos.

Casos de Teste
Caso	Entrada	Resultado Esperado
CT01	Campos obrigatórios vazios	Todos os campos são obrigatórios
CT02	Todos os campos preenchidos corretamente	Vacina registrada com sucesso
